# Vanilla Web Video Recorder Experiments

This repository is for experimenting with a **web-based video recorder** in plain HTML/CSS/JavaScript.

## Goal
Build and test recording approaches that make browser-captured video quality as close as possible to the quality of built-in mobile camera/video recorder apps.

## Focus Areas
- Camera constraints and device compatibility (especially mobile)
- Codec/container support across browsers
- Bitrate and quality tuning
- Preview vs recorded output consistency
- UX improvements for reliable mobile recording
